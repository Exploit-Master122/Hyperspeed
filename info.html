Darrkon:

Darrkon is an amazing web proxy that acts as a leech for other wisp servers. While not the most reliable for support of lots of websites, the concept of a static hosting proxy is great.

Proxy Backends

Darrkon supports multiple proxy backends to route your traffic:

Ultraviolet Working

A highly sophisticated proxy used for evading internet censorship. It's currently the recommended backend for optimal performance and compatibility.

ScramJet Currently Broken

The most reliable proxy that supports lots of sites and web apps, and runs almost the entirety of the web. Unfortunately, ScramJet is down.

WISP Server

WISP (WebSocket Internet Service Protocol) is used to establish secure connections through the proxy. The default server is wss://anura.pro/ and should work for most users.

You can change the wisp server if you know what you're doing or you have your own.
